# CalculadoraFisica-Giovana
Calculadora de Queda Vertical
Projeto de Física - Java

Descrição: > Este programa resolve problemas de queda livre e lançamento oblíquo descendente.
Baseado no exercício de uma pedra lançada a 12 m/s de uma altura de 30m.  

Como usar:

Insira a velocidade, altura e ângulo (0 a -90°).
2. Clique em Calcular para obter o tempo (Bhaskara) e a velocidade final.
